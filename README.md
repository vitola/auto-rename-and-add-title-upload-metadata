# Alex Vitola Upload Metadata

Alex Vitola Upload Metadata is a lightweight WordPress plugin that automatically organizes image uploads.

## Features

- Automatically renames uploaded images using a unique timestamp.
- Automatically sets the media title from the filename.
- Automatically fills the media caption from the embedded copyright metadata.
- Lightweight and configuration-free.
- Uses native WordPress APIs.

## Requirements

- WordPress 6.0+
- PHP 7.4+
